# Модуль векторной алгебры на C++ с интеграцией в Python (Pybind11)

Практическая работа по созданию гибридного Python-модуля на базе компилируемого C++ кода с использованием библиотеки **Pybind11**.

## Структура проекта
* `vector_module.cpp` — исходный код на C++ (функции векторной алгебры и класс `Vector3D`).
* `setup.py` — скрипт сборки модуля средствами Python `setuptools`.
* `CMakeLists.txt` — конфигурационный файл для сборки проекта через систему CMake.
* `test_vector.py` — юнит-тесты на Python для проверки работы кода.

## Требования
* Компилятор C++ с поддержкой стандарта C++11 (например, `gcc` / `g++`)
* Python 3.x
* Установленный пакет `pybind11` (`pip install pybind11`)
* Система сборки `CMake` (опционально)

## Инструкция по сборке и запуску

### Вариант 1: Сборка через Python (setup.py)
Для сборки и установки модуля в текущее окружение в режиме разработки выполните команду:
```bash
pip install -e .
```

### Вариант 2: Локальная сборка через CMake
```bash
mkdir build && cd build
cmake ..
make
cp *.so ..
cd ..
```

## Тестирование
После успешной сборки запустите проверочные тесты:
```bash
python3 test_vector.py
```

При успешном выполнении тестов в консоли отобразится статус `OK`.
